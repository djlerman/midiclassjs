Library Design

The MIDIFile module

  • Always contains raw message data; unprocessed in any way.

The MIDISequence module

  • Event parameters are interpreted to be human readable whenever possible.
  • Many of the interpretation routines are contained inside the Utils module for use in the the text and xml import modules.
