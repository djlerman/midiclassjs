addEvent()

Adds the given event to this track. Checks for all required parameters for the event's type and throws an error if any are missing. Furthermore, defines index properties for these parameters in their defined ordering as per the MIDI specification.

replaceEvent()

Replaces the event at index i with evt; evt's parameters are checked as if it were added with addEvent. Throws an error if the parameters are invalid, and preserves current track events.
